This is a free, multi-page school website built with HTML/CSS/JS intended for Government Primary School Pethgam Wagoora.

## Features
- Header with contact info and UDISE
- Scrolling announcement ticker
- 3D-styled live menu buttons
- Hero slider
- Quick links and news/events area
- Principal message, animated counters
- Gallery and contact form
- Mobile-friendly, ready to host on GitHub Pages

## How to use
1. Replace images in `assets/images/` (logo.png, hero images, gallery images).
2. Edit contact info and page content as needed.
3. Push to GitHub and enable GitHub Pages (use `main` branch root) to publish.

## Notes
- Contact form and application form are front-end demos. To accept submissions, integrate with a back-end or service (e.g., Google Forms, Formspree, Firebase).
